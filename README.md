# 🧬 GenomeLab: Structural Evolution Engine (Phase 2)

GenomeLab is an adaptive bioinformatics platform designed to simulate genetic mutations and evaluate their impact on protein stability and structural realism.

## 🚀 Phase 2 Features: Physical Realism
This version moves beyond simple text manipulation to simulate real-world biochemical stressors:
* **Secondary Structure Prediction:** Calculates Alpha-Helix and Beta-Sheet fractions.
* **Environmental Stress Testing:** Simulates protein survival across varying **Temperature (0-100°C)** and **pH (0-14)** scales.
* **Biophysical Profiling:** Tracks Isoelectric Point (pI) and Molecular Weight (kDa).
* **Natural Selection API:** A FastAPI-powered backend that handles evolutionary iterations.

## 🛠️ Tech Stack
* **Backend:** Python 3.12, FastAPI, Biopython, Pandas
* **Deployment:** Render (Backend API), Vercel (React Frontend)
* **Simulation Engine:** Streamlit (Internal Prototyping)
